# 💰 个人资产管理和投资记录工具

一个简单易用的个人财务管理应用，帮助你记录和分析你的资产、投资和股票持仓。

## 🌟 功能特性

### 📊 多维度数据图表
- **现金流趋势**：折线图展示现金存款随时间的变化
- **总资产增长**：折线图追踪总资产的增长曲线
- **资产配置**：饼图直观展示各类资产的占比分布

### 📈 增长率分析
- **复合增长率（CAGR）**：自动计算从首条记录到最新记录的年化复合增长率，以仪表盘图表展示（0–100% 范围），红黄绿三色直观反映增长水平
- **月环比增长率**：柱状图展示每月相对于上月的增长百分比，正增长显示粉色，负增长显示绿色，标签标注具体数值

### 📝 灵活的数据输入
- 平安银行账户管理
- 平安证券账户管理
- 支付宝账户管理
- 农业银行账户管理
- 基金市值记录（平安银行基金、平安证券基金、支付宝基金、雪球基金）
- 顺丰控股股票持仓

### 🗂️ 数据表格标签页
历史记录区域提供四个标签页，便于从不同维度查看数据：
- **账户总值**：各银行和平台的账户余额
- **基金市值**：各平台的基金持仓市值
- **股票市值**：股票持仓市值
- **会计统计**：自动计算的现金流、基金资产、股票资产和总资产

### 💾 数据安全与备份
- **本地自动保存**：数据实时保存至浏览器本地存储，无需服务器
- **导出备份**：一键将全部数据导出为带日期的 JSON 备份文件（如 `finance-tracker-backup-2026-04-15.json`），防止浏览器缓存清除导致数据丢失
- **导入恢复**：支持从 JSON 备份文件恢复数据，可选择**覆盖**或**合并**两种模式

### 👤 用户头像
- 点击页面顶部的"+"圆形区域上传个人头像
- 头像自动保存至浏览器本地存储，刷新页面后依然保留

### 🎨 美观的界面设计
- 响应式设计，支持各种屏幕尺寸
- 现代化渐变色与动画效果

## 📖 使用说明

### 添加新记录
1. 填写"日期"（选择年月）
2. 在"账户总值"区域填入各银行/平台的账户余额
3. 在"投资资产"区域填入各基金市值和股票市值
4. 点击"添加记录"按钮
5. 数据自动保存，图表和统计实时更新

### 查看历史记录
- 点击"历史记录"区域上方的标签页切换不同维度的数据视图
- **会计统计**标签页自动计算：
  - **现金流** = 各账户总值之和 − 基金市值之和
  - **基金资产** = 所有基金市值之和
  - **股票资产** = 顺丰控股市值
  - **总资产** = 现金流 + 基金资产 + 股票资产

### 删除记录
- 在任意标签页的表格中，点击对应行最右侧的"删除"按钮即可

### 导出数据备份
1. 点击表单下方的"导出数据"按钮
2. 浏览器自动下载 JSON 备份文件（文件名含当天日期）
3. 妥善保存该文件，可随时用于恢复数据

### 导入数据恢复
1. 点击"导入数据"按钮
2. 选择之前导出的 `.json` 备份文件
3. 在弹出提示中选择：
   - **确定（覆盖）**：用备份数据完全替换当前数据
   - **取消（合并）**：将备份数据与当前数据合并，重复日期以备份为准

### 上传个人头像
1. 点击页面顶部左侧的"+"圆形区域
2. 选择本地图片文件
3. 头像即时显示，并自动保存至浏览器本地存储

### 查看增长率
- **CAGR 仪表盘**：表单卡片下方自动显示复合年化增长率，颜色由红到绿反映增长水平
- **月环比柱状图**：展示每月环比增长率，正增长（粉色）和负增长（绿色）一目了然

## 🛠️ 技术栈

## 📌 注意事项

- 数据存储在浏览器本地，**清除浏览器缓存会导致数据丢失**，请定期使用"导出数据"功能备份
- 建议使用 Chrome / Edge / Firefox 等现代浏览器以获得最佳体验
