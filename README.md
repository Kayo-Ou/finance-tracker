# 💰 个人资产管理和投资记录工具

一个简单易用的个人财务管理应用，帮助你记录和分析你的资产、投资和股票持仓。

## 🌟 功能特性

- 📊 **三个实时图表**
  - 现金存款变化趋势（折线图）
  - 资产占比分布（饼图）
  - 总资产变化趋势（折线图）

- 📝 **灵活的数据输入**
  - 支付宝账户管理
  - 平安银行账户管理
  - 蛋卷基金记录
  - 平安证券管理
  - 农业银行账户管理
  - 顺丰控股持仓

- 💾 **本地数据保存**
  - 浏览器自动保存（不用担心数据丢失）
  - 无需服务器，完全离线使用

- 🎨 **美观的界面设计**
  - 响应式设计，支持各种屏幕
  - 现代化的渐变色和动画效果

## 🚀 快速开始

### 方式 1：直接在浏览器打开
1. 下载这个项目到你的电脑
2. 找到 `index.html` 文件
3. 双击打开即可使用

### 方式 2：用 GitHub Pages 在线使用
1. Fork 这个项目
2. 进入 Settings → Pages
3. 选择 "Deploy from a branch"
4. 选择 `main` 分支
5. 等待几秒，你就能在线使用了！

## 📖 使用说明

### 添加新记录
1. 填写"统计日期"（格式：YYYY-MM）
2. 填入各个账户的金额
3. 点击"➕ 添加记录"按钮
4. 数据会自动保存，图表实时更新

### 查看历史记录
- 表格显示所有月份的详细数据
- 自动计算：
  - **现金存款** = 支付宝 + 平安银行 + 农业银行 - 基金部分
  - **投资基金** = 所有基金之和
  - **持有股票** = 平安证券股票 + 顺丰控股
  - **总资产** = 所有资产总和

### 删除记录
- 点击表格最右边的"删除"按钮即可
